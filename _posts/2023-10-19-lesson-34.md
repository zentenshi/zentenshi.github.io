## Sentence Pattern 1
```
(Verb, た形)とおりに, (Verb 2)
```

## Sentence Pattern 2
```
(Verb, た形)後で (Verb 2)
(Noun)の後で (Verb 2)
```
Do Verb 2 with/out doing Verb 1

## Sentence Pattern 3
```
(Verb, て形) (Verb 2)
(Verb, ない形)で (Verb 2)
```
Do Verb 2 without doing Verb 1

Note that you can only use this with volitional verbs

## Sentence Pattern 4
```
(Verb, て形) (Verb 2)
(Verb, ない形)で (Verb 2)
```
Do Verb 2 instead of Verb 1
