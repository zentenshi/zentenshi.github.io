## Sentence Pattern 1
```
(Interrogative)　(Verb・Adj・Noun, 普通形)か　～
```

## Sentence Pattern 2
```
(Verb・Adj・Noun, 普通形)かどうか　～
```
For なAdj and Nouns in present positive tense, `だ` is removed

## Sentence Pattern 3
```
(Verb, て形)　みます
```
This SP is not limited to `ーます`
